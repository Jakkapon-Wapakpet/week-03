# Chrome & Burger - DB Querying & Application Assessment

This repository contains the Database Querying & Application Assessment for **Chrome & Burger**. It includes database initialization scripts, query tasks (MongoDB & PostgreSQL), a static landing page, and an Express API server.

## 🔗 Live Deployment Links
* **Frontend (Vercel):** [https://chrome-burger-app.vercel.app/menu.html](https://chrome-burger-app.vercel.app/menu.html)
* **Backend API Server (Render):** [https://chrome-burger-api.onrender.com](https://chrome-burger-api.onrender.com)

---

## 📁 Repository Structure

* **`01_initialize_databases`**: Scripts to set up and seed tables/collections for MongoDB and PostgreSQL.
* **`02_querying_assessment`**: Assessment query tasks for both MongoDB and PostgreSQL, including thinking processes and SQL/NoSQL aggregate queries.
* **`03_chrome-burger-landing-page`**: Static landing page built with HTML and Tailwind CSS, fetching live menu items from the API backend.
* **`04_example_api-server`**: Express API server connecting to both MongoDB and PostgreSQL databases to serve menu items.
