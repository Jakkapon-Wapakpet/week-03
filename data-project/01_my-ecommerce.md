# 🎮 GearHub - Smart Gaming Gear Store

ยินดีต้อนรับสู่เอกสารโครงการเริ่มต้นของ **GearHub** แพลตฟอร์มอีคอมเมิร์ซแบบผู้ขายรายเดียว (Single-vendor E-commerce) ที่มุ่งเน้นการปฏิวัติประสบการณ์การซื้ออุปกรณ์เกมมิ่งเกียร์ (Gaming Peripherals) ให้ง่าย สะดวก และคุ้มค่าที่สุดผ่านนวัตกรรมระบบเปรียบเทียบสเปกในตัว

---

## 📌 1. ที่มาและความสำคัญของโครงการ (Background & Problem Statement)

ในปัจจุบัน อุปกรณ์เกมมิ่งเกียร์ เช่น เมาส์ คีย์บอร์ด และหูฟัง มีความหลากหลายและพัฒนาไปอย่างรวดเร็ว ส่งผลให้ตลาดมีสินค้าที่มีรูปลักษณ์ ราคา และสเปกทางเทคนิคที่ใกล้เคียงกันเป็นจำนวนมาก ทำให้ผู้ใช้ประสบปัญหาดังนี้:
* **ความสับสนในเรื่องสเปกเชิงเทคนิค:** เช่น Switch คีย์บอร์ดต่างสีต่างแบรนด์, เซนเซอร์เมาส์, หรือ Polling Rate ของเมาส์ไร้สาย
* **ความยุ่งยากในการเปรียบเทียบ:** ร้านค้าออนไลน์ส่วนใหญ่จะนำเสนอข้อมูลแบบแยกชิ้น ทำให้ผู้ซื้อต้องเปิดแท็บเว็บเปรียบเทียบด้วยตัวเองหลายหน้าต่าง ส่งผลให้เกิดความเหนื่อยล้าในการค้นหาข้อมูล (Choice Overload)
* **การขาดข้อมูลสำหรับกลุ่มที่ไม่ใช่เกมเมอร์:** กลุ่มที่ซื้อของขวัญให้ลูกหลานไม่สามารถวิเคราะห์ได้ว่าอุปกรณ์ชนิดใดทำงานร่วมกับอุปกรณ์ที่ผู้รับมีอยู่ได้บ้าง

**GearHub** จึงถูกพัฒนาขึ้นเพื่อลดช่องว่างเหล่านี้ โดยการนำเสนอหน้าเว็บที่สะอาด สวยงาม ทันสมัย และมีระบบ **Product Comparison (เปรียบเทียบคุณสมบัติสินค้าแบบเคียงข้างกัน)** ที่ทรงพลัง เพื่อช่วยให้ทุกคนซื้ออุปกรณ์ที่เหมาะสมกับงบประมาณและความต้องการของตนเองได้อย่างอุ่นใจและรวดเร็ว

## 💎 2. คุณค่าหลักของระบบ (Unique Value Proposition - UVP)

1. **Built-in Smart Comparison:** ระบบเปรียบเทียบสเปก วัสดุ น้ำหนัก และราคาสินค้าได้สูงสุด 3 รุ่นพร้อมกันในคลิกเดียว พร้อมไฮไลท์จุดเด่นเฉพาะตัว (เช่น เบาที่สุด หรือ คุ้มที่สุด)
2. **Simplified Technical Specs:** แปลข้อมูลจำเพาะเชิงลึกให้เป็นไอคอนและการแสดงผลที่เข้าใจง่าย เหมาะสมกับทั้งเกมเมอร์สายแข่งไปจนถึงผู้ปกครองที่ไม่มีความรู้ไอที
3. **Seamless Social Connection & Tracking:** เชื่อมต่อการชำระเงินด่วน (Instant Scan QR Payment) และอัปเดตสถานะของพัสดุรวมถึงรับใบประกันสินค้าผ่านระบบ LINE Official Account เพื่อสร้างประสบการณ์ช็อปปิ้งแบบไร้รอยต่อ

## 👥 4. กลุ่มเป้าหมายผู้ใช้งาน (Target User Personas)

รายละเอียดกลุ่มผู้ใช้ตัวแทน 4 รูปแบบหลักที่มีอิทธิพลต่อฟังก์ชันและดีไซน์ของ GearHub:

### 4.1 พีท (Pete) - Esports Gamer
> "อุปกรณ์ทุกชิ้นต้องมี Latency ต่ำที่สุด เมาส์ต้องเบา สวิตช์คีย์บอร์ดต้องตอบสนองทันที!"
* **ข้อมูลทั่วไป (General Info):** อายุ 22 ปี เป็นนักศึกษาและนักแข่งเกมนอกเวลา เล่นเกมแนว FPS (VALORANT, Apex) วันละ 6-8 ชม.
* **ความต้องการหลัก (Goals):** ตารางเปรียบเทียบขนาด น้ำหนัก สเปกเซนเซอร์ และ Polling Rate ไร้สายที่ลึกและละเอียด ของแท้จัดส่งด่วนมีประกันไทย
* **ปัญหาที่เจอ (Pain Points):** ร้านค้าส่วนใหญ่ไม่แจ้งน้ำหนักเมาส์หรือชิปเซนเซอร์ข้ามแบรนด์ ทำให้เทียบข้ามแบรนด์ได้ยาก
* **Journey (เส้นทางการซื้อ):** 
  * *Awareness:* เมาส์ตัวเดิมดับระหว่างแข่งซ้อม อยากได้ตัวใหม่ที่เบากว่าเดิมด่วน
  * *Consideration:* ใช้ฟีเจอร์ Compare ของ GearHub เทียบ Razer Viper V3 Pro กับ Logitech GPX2
  * *Decision:* เลือกสินค้าที่สเปกตรงความเร็วที่ต้องการ จ่ายผ่านระบบพร้อมเพย์ที่รวดเร็ว
  * *Post-Purchase:* ได้รับของเร็วภายใน 1 วัน ลองปรับโปรแกรมความเร็วและใช้แข่งซ้อมได้ทันที

### 4.2 ฟ้า (Fah) - Aesthetic & Casual Gamer
> "อยากจัดโต๊ะคอมโทนสีพาสเทล คีย์บอร์ดต้องสวยงามและเสียงกดต้องนุ่มหู (Thocky)"
* **ข้อมูลทั่วไป (General Info):** อายุ 19 ปี นักศึกษาอักษรศาสตร์ เล่นเกมแนวผ่อนคลาย (Genshin, Stardew Valley, Sims)
* **ความต้องการหลัก (Goals):** ค้นหาคีย์บอร์ด เมาส์ โทนสีพาสเทล (ชมพู/ขาว) ราคาเหมาะสม ทนทาน และใช้คำอธิบายศัพท์ไอทีภาษาไทยง่ายๆ
* **ปัญหาที่เจอ (Pain Points):** อุปกรณ์เกมมิ่งเกียร์สไตล์ทั่วไปดุดันและมีแต่สีดำ ศัพท์เทคนิคเยอะเข้าใจยาก
* **Journey (เส้นทางการซื้อ):**
  * *Awareness:* เห็นการแต่งโต๊ะทำงานสีพาสเทลน่ารักบน TikTok เลยอยากมีของตนเองบ้าง
  * *Consideration:* เข้าเว็บ GearHub กรองหา "สีชมพู" และมีปุ่มเปรียบเทียบ/คำอธิบายประเภทสวิตช์ง่ายๆ
  * *Decision:* ตัดสินใจสั่งซื้อคีย์บอร์ด Linear Red Switch ราคา 1,800 บาท ชำระเงินด้วยบัตรเดบิต
  * *Post-Purchase:* ของตรงปกสีสวยงาม ถ่ายรูปลงสตอรี่ไอจีและเอามาใช้พิมพ์รายงานได้เป็นอย่างดี

### 4.3 ก้อง (Kong) - PC & Tech Enthusiast
> "ผมยินดีจ่ายแพงกว่าเพื่อวัสดุระดับพรีเมียม และระบบเชื่อมต่อที่เสถียรไร้ที่ติ"
* **ข้อมูลทั่วไป (General Info):** อายุ 28 ปี (Software Engineer) ชอบโมดิฟายด์ สะสม Custom Keyboards และคอลเลกชัน Limited
* **ความต้องการหลัก (Goals):** คีย์บอร์ดบอดี้ CNC อะลูมิเนียม เชื่อมต่อ 3 โหมด ข้อมูลวัสดุภายในที่ละเอียด และบริการดูแลหลังการขายแบบ VIP
* **ปัญหาที่เจอ (Pain Points):** สินค้าพรีเมียมของหมดไวในไทย หรือต้องหารายละเอียดโครงสร้างบอดี้/สาร Lube สวิตช์ยาก
* **Journey (เส้นทางการซื้อ):**
  * *Awareness:* ทราบข่าว Keychron รุ่นบอดี้อะลูมิเนียมนำเข้ามาใหม่ในไทยจากกลุ่ม Discord
  * *Consideration:* เข้า GearHub ใช้ตัวกรองกลุ่ม High-End ดูเปรียบเทียบวัสดุเพลตและประเภท Gasket
  * *Decision:* สั่งซื้อรุ่นท็อป 5,900 บาท ใช้ระบบผ่อนชำระ 0% พร้อมลงทะเบียนสมาชิก VIP เพื่อรับบริการส่งด่วนในวันนั้น
  * *Post-Purchase:* ได้รับของด่วน นำมาถอดอัปเกรดสวิตช์เพิ่มเติม และแบ่งปันในคอมมูนิตี้คนจัดโต๊ะคอม

### 4.4 แม่วรรณ (Mae Wan) - Gift Buyer (Non-Gamer)
> "แม่ไม่รู้เรื่องไอทีหรอก รู้แค่ลูกอยากได้หูฟังใส่สบาย ใช้เรียน เล่นเกม ทนทาน ไม่แพงเกินไป"
* **ข้อมูลทั่วไป (General Info):** อายุ 45 ปี พนักงานบริษัทเอกชน ต้องการหาของขวัญวันเกิดชิ้นพิเศษให้ลูกชาย
* **ความต้องการหลัก (Goals):** เว็บไซต์ที่สั่งซื้อสะดวก ไม่ต้องมีขั้นตอนซับซ้อน ป้ายบอกสินค้าขายดี และระบุความเข้ากันได้กับเครื่องเกมลูก
* **ปัญหาที่เจอ (Pain Points):** ไม่เข้าใจคำศัพท์ เช่น 7.1 Virtual, Latency, Driver และกลัวซื้อมาผิดขั้วต่อเชื่อมต่อใช้ไม่ได้
* **Journey (เส้นทางการซื้อ):**
  * *Awareness:* ลูกชายบ่นหูฟังเดิมชำรุด อยากหาของขวัญวันเกิดให้ในงบ 2,000-3,000 บาท
  * *Consideration:* ค้นเว็บ GearHub เลือกหมวดสินค้าขายดี/ของขวัญ ดูไอคอนยืนยันว่าเสียบเล่นกับ PS5/PC ได้
  * *Decision:* สั่งซื้อรุ่นยอดนิยมสัญลักษณ์ 'ขายดีอันดับ 1' ราคา 2,400 บาท แบบผู้มาเยือน (Guest Checkout) โดยไม่ต้องสมัครสมาชิก
  * *Post-Purchase:* ของส่งถึงบ้านใน 2 วันทันวันเกิด ห่อของกันกระแทกดี ลูกชายได้รับใช้ได้ทันทีไม่มีปัญหา

## 🛠️ 7. สถาปัตยกรรมและเทคโนโลยีที่ใช้ (Tech Stack)

* **Frontend:** React, Vite, Tailwind CSS (เพื่อการพัฒนา UI ที่สวยงาม โหลดเร็ว และเป็น Responsive เต็มรูปแบบ)
* **Backend:** Node.js, Express.js (จัดการระบบ RESTful API ความเร็วสูง)
* **Database:** MongoDB (จัดเก็บข้อมูลผู้ใช้ รายการสินค้า และข้อมูลคำสั่งซื้อในรูปแบบ JSON Documents ที่มีความยืดหยุ่นสูง)
* **Authentication:** JSON Web Token (JWT) และ OAuth (สิทธิ์ความปลอดภัยการล็อคอิน)
